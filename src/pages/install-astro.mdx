## Requeremientos

```go
- GitHub
- Node JS 18.10 o superior
- Visual Studio code con la extensión de Astro
```


1.- Como *primer* paso debemos instalar astro en nuestra computadora con el siguiente comando: (Puedes consultar más info en )
[Astro Docs](https://astro.build/)
```js
npm create astro@latest
```

2.- Acceder al directorio con el siguiente comando:
```js
cd blog-volta
```
